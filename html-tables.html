Learing HTML Tables by Creating School Time Table

This page is all about learning HTML tables by taking school time table as a use case. While learing to use these tables, I came accross lot of other HTML functionalities which had to be used in this same document. I have appended the same in the learnings section.

Each section is divided into Objectives, Output and Learnings.

Objectes:

Create a simple school time table using only the basic HTML tags.

Learnings:

  • Caption tag for tables.
  • Basic border attribute for tables.
  • Table Header (th) tag inside Table Row Element (tr) for highlighting the column as header.
  • What are Table Head (thead) and Table Body(tbody) tags.
  • How to use Abbreviation (abbr) tags
  • Tags vs Elements vs Attributes in HTML
    1. Tags: Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags.
    2. Elements: Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag.
    3. Attributes: : It is used to define the character of an HTML element. It always placed in the opening tag of an element. It generally provides additional styling (attribute) to the element.
      4. GeeksforGeeks
  • Ordered (ol) and Unordered List (ul) tags.
  • Cite tag and basic a tag with href.

Learnings:

  • colspan and rowspan:
    1. colspan: a td element which denotes how many columns to span across. It takes in an integer.
    2. rowspan: a td element which denotes how many row to span across. It takes in an integer. However, a rowspan value of 0, denotes that the row should span till the end.
